# 🍔 FoodFlie

FoodFlie is a modern food delivery platform designed to provide fast, reliable, and seamless ordering experiences for users.

## 🚀 Overview

FoodFlie allows users to browse restaurants, place orders, and receive their food efficiently. The platform is being built with scalability and user experience in mind.

## 🎯 Features

* User authentication with OTP verification
* Browse restaurants and menus
* Add items to cart and place orders
* Real-time order tracking (planned)
* Delivery partner assignment system
* Admin dashboard for store management

## 🔐 Authentication

FoodFlie uses OTP-based authentication for secure and quick user login.
Integration with Twilio is used to send OTPs to users for verification.

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL
* **APIs:** REST APIs
* **Messaging/OTP:** Twilio
